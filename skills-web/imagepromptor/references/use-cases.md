## Case 1: PRODUCT / E-COMMERCE üì¶

### Detection Keywords
`produit`, `e-commerce`, `packshot`, `boutique`, `shop`, `vente`, `article`

### Characteristics
- Neutral or contextual background
- Macro or medium shot
- Studio lighting (soft, controlled)
- Focus on materials and finish

### Default Parameters
| Parameter | Default Value |
|-----------|---------------|
| Style | photorealistic |
| Lighting | soft diffused studio light |
| Composition | 50-100mm macro, f/2-2.8, centered |
| Ratio | 1:1 (Instagram) or 4:3 (web) |
| Quality | 4K professional quality, premium studio render |
| Negatives | shadows, reflections, fingerprints, dust, blur, text |

### Template
```
A [material] [product] in [finish/color] with [specific details],
photorealistic,
soft diffused studio light on [background color/surface],
[focal]mm product shot f/[aperture], centered composition on [surface],
4K professional quality, premium studio render,
Negative: shadows, reflections, fingerprints, dust, blur, text, watermark
```

### Example
```
A premium wireless headphone in midnight black with rose gold accents,
photorealistic,
soft diffused studio light on white backdrop,
50mm product shot f/2, centered composition on white surface,
4K professional quality, premium studio render,
Negative: shadows, reflections, extra parts, blur, sketch style
```

## Case 6: TEXT / LOGO üìù

### Detection Keywords
`texte`, `logo`, `affiche`, `banni√®re`, `titre`, `label`, `marque`

- **Explicit placement**: "top-center", "bottom-right", etc.

## Case 7: IMAGE-TO-IMAGE üîÑ

### Detection Keywords
`modifier`, `changer`, `remplacer`, `√©diter`, `transformer`, `ajuster`

### Characteristics
- Focus on SPECIFIC changes only
- Preserve original elements
- Don't rewrite entire scene

### Critical Rules
- Start with preservation instructions
- List only what changes
- Be specific about modifications

### Template
```
Keep original [elements to preserve],
replace [element] with [new element],
change [aspect] to [new state],
add [new elements],
maintain [qualities to keep],
[style],
Negative: [unwanted changes], unnatural transitions, washed colors
```

### Example
```

replace background with sunset ocean view,
change lighting to warm golden hour backlight,
add soft rim lighting,
maintain original subject details,
photorealistic, cinematic,
Negative: distorted subject, unnatural transitions, washed colors
```

### Detection Keywords
`s√©rie`, `variations`, `d√©clinaisons`, `coh√©rent`, `plusieurs`, `collection`

### Characteristics
- Base prompt stays identical
- Only specified elements vary
- Maintain character/subject consistency

### Critical Rules
- Define fixed base (character traits, core style)
- Explicitly mark variable elements
- Assign reference roles if using images

### Template Structure
```

[fixed style],
[VARIABLE: lighting, background, pose, clothing]

VARIATION 1: [specific changes]
VARIATION 2: [specific changes]

```
