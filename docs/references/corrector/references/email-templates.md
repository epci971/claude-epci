# Email Templates Reference

> Structure templates organized by email intent/purpose

---

What is the email trying to achieve?

## 1. Update / Information Email

### Purpose
Share information, progress, or status without requiring action.

### Structure
```
GREETING

CONTEXT (1-2 sentences)
- Reference to project/topic
- Why this update now

CONTENT
- Key information (what happened, what changed)
- Relevant details (organized logically)
- Impact or implications if any

NEXT STEPS (optional)
- What happens next
- When next update expected

CLOSING
- Availability for questions

SIGN-OFF + SIGNATURE
```

### Trigger phrases
- "Je voulais vous tenir informé..."
- "Petit point sur..."
- "Suite à..."
- "Pour info..."

### Example skeleton (French)
```
Bonjour [Destinataire],

Suite à [contexte/événement], je vous fais un point sur [sujet].

[Contenu principal - informations clés]

[Prochaines étapes si pertinent]

N'hésitez pas si vous avez des questions.

[Formule de politesse]

[Signature]
```

## 2. Request Email

### Purpose
Ask for something: information, action, resource, meeting.

### Structure
```
GREETING

CONTEXT (brief)
- Why you're reaching out
- Reference if applicable

REQUEST (clear and specific)
- What you need
- Why you need it (justification)
- Deadline if any

DETAILS (if needed)
- Specifications
- Constraints
- Options if flexible

FACILITATION
- What you can provide to help
- Your availability

CLOSING
- Thank you in advance
- Availability for discussion

SIGN-OFF + SIGNATURE
```

### Trigger phrases
- "J'aurais besoin de..."
- "Serait-il possible de..."
- "Pourriez-vous..."
- "Je me permets de vous solliciter..."

### Example skeleton (French)
```
Bonjour [Destinataire],

Dans le cadre de [contexte], j'aurais besoin de [requête spécifique].

[Justification / pourquoi c'est nécessaire]

[Détails ou spécifications si besoin]

Idéalement, ce serait pour [date/délai]. Je reste disponible pour en 
discuter si nécessaire.

Merci d'avance pour votre retour.

[Formule de politesse]

[Signature]
```

## 3. Validation / Approval Email

### Purpose
Get sign-off, agreement, or approval on something.

### Structure
```
GREETING

CONTEXT
- What needs validation
- Reference to previous discussions

SUMMARY OF PROPOSAL
- Clear statement of what you're proposing
- Key points (bullet format acceptable here)

RATIONALE (brief)
- Why this approach
- Alternatives considered if relevant

ASK (explicit)
- Clear request for validation
- What form of response you need

NEXT STEPS
- What happens after approval
- Timeline

SIGN-OFF + SIGNATURE
```

### Trigger phrases
- "Je soumets à votre validation..."
- "Merci de me confirmer..."
- "J'attends votre feu vert pour..."
- "Pourriez-vous valider..."

### Example skeleton (French)
```
Bonjour [Destinataire],

Suite à nos échanges, je vous soumets [proposition] pour validation.

En résumé :
- [Point clé 1]
- [Point clé 2]
- [Point clé 3]

[Justification brève si nécessaire]

Merci de me confirmer votre accord pour que je puisse lancer [prochaine étape].

[Formule de politesse]

[Signature]
```

## 4. Follow-up / Reminder Email

### Purpose
Remind about pending request, check status, get update.

### Structure
```
GREETING

REFERENCE (non-accusatory)
- What you're following up on
- When original request was made

CONTEXT REMINDER (brief)
- Why it matters
- Current situation

SOFT ASK
- Request for status
- Offer to help if blocked

FLEXIBILITY
- Understanding of constraints
- Alternative timeline if needed

SIGN-OFF + SIGNATURE
```

### Trigger phrases
- "Je me permets de revenir vers vous..."
- "Suite à mon message du..."
- "Avez-vous eu l'occasion de..."
- "Je voulais faire un point sur..."

### Example skeleton (French)
```
Bonjour [Destinataire],

Je me permets de revenir vers vous concernant [sujet] suite à mon 
message du [date].

[Rappel bref du contexte/besoin]

Avez-vous eu l'occasion d'avancer sur ce point ? N'hésitez pas à me 
faire signe si vous avez besoin d'éléments complémentaires de mon côté.

[Formule de politesse]

[Signature]
```

## 5. Response / Answer Email

### Purpose
Reply to a request, question, or inquiry.

### Structure
```
GREETING

ACKNOWLEDGMENT
- Thank for the message/request
- Confirm understanding

ANSWER (direct)
- Response to the main question
- Decision if applicable

DETAILS
- Supporting information
- Explanations if needed

NEXT ACTIONS (if any)
- What happens now
- Who does what

AVAILABILITY
- Open for follow-up questions

SIGN-OFF + SIGNATURE
```

### Trigger phrases
- "Suite à votre demande..."
- "En réponse à..."
- "Pour faire suite à votre message..."
- "Concernant votre question..."

### Example skeleton (French)
```
Bonjour [Destinataire],

Merci pour votre message concernant [sujet].

[Réponse directe à la question/demande]

[Détails ou explications complémentaires]

[Prochaines étapes si pertinent]

Je reste disponible si vous avez d'autres questions.

[Formule de politesse]

[Signature]
```

## 6. Meeting Request Email

### Purpose
Propose or schedule a meeting.

### Structure
```
GREETING

PURPOSE
- Why you want to meet
- What you want to discuss/achieve

PROPOSED LOGISTICS
- Suggested date(s)/time(s)
- Duration estimate
- Format (call, video, in-person)

PREPARATION (if needed)
- What to bring/prepare
- Documents to review beforehand

FLEXIBILITY
- Alternative if proposed times don't work
- Your availability constraints

SIGN-OFF + SIGNATURE
```

### Example skeleton (French)
```
Bonjour [Destinataire],

Je souhaiterais organiser un point avec vous concernant [sujet] afin de 
[objectif].

Je vous propose [date/heure], pour environ [durée]. Le format serait 
[visio/téléphone/présentiel].

[Préparation éventuelle]

N'hésitez pas à me proposer un autre créneau si celui-ci ne vous 
convient pas.

[Formule de politesse]

[Signature]
```

## 7. Bad News / Sensitive Email

### Purpose
Communicate negative information, delay, problem, or rejection.

### Structure
```
GREETING

POSITIVE OPENING (if genuine)
- Acknowledgment of relationship/effort
- Context appreciation

NEWS (direct but empathetic)
- State the situation clearly
- Don't bury the lead

EXPLANATION (brief)
- Why this happened
- Factors involved (without excuses)

SOLUTION/MITIGATION
- What you're doing about it
- Alternatives offered
- How you'll prevent recurrence

COMMITMENT
- Your dedication to resolution
- Next steps

SIGN-OFF + SIGNATURE
```

### Key principles
- Be direct, don't bury bad news
- Take responsibility where appropriate
- Always offer path forward
- Empathy without excessive apology

### Example skeleton (French)
```
Bonjour [Destinataire],

Je reviens vers vous concernant [projet/sujet].

Malheureusement, [mauvaise nouvelle - directe et claire].

[Explication brève - factuelle]

Pour pallier cette situation, je vous propose [solution/alternative]. 
[Actions concrètes que vous prenez]

Je reste à votre disposition pour en discuter et trouver ensemble la 
meilleure solution.

[Formule de politesse]

[Signature]
```

## Template Adaptation Rules

### Combine templates when needed
- Update + Request = "Here's where we are, and I need X to continue"
- Response + Follow-up = "Here's my answer, and what about Y?"

### Length guidelines by context

| Context | Target Length |
|---------|---------------|
| Quick internal update | 3-5 sentences |
| Client status update | 1-2 short paragraphs |
| Formal request | 2-3 paragraphs |
| Sensitive/complex topic | As needed, but structured |

### When to use bullet points
- Multiple items to list
- Technical specifications
- Action items
- Decision points

### When to avoid bullet points
- Emotional/sensitive content
- Relationship-building messages
- Simple communications
- Formal correspondence
