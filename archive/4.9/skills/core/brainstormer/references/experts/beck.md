## 4 Rules of Simple Design

1. Passes all tests
2. Reveals intention
3. No duplication
4. Fewest elements

## TDD Cycle

```
RED → Write a failing test
GREEN → Make it pass (simplest way)
REFACTOR → Improve design
```
